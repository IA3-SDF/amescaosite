## 📌 À propos du projet

Ceci est l'application web officielle de l'**AMESCAO** (Amicale des Élèves, Étudiants et Stagiaires du Canton de Aouda). Conçue comme une **PWA** (Progressive Web App), elle offre une expérience mobile premium, un support hors-ligne et des fonctionnalités d'interaction communautaire.

## 🛠️ Lancement en Local

Suivez ces étapes simples pour configurer et lancer le projet sur votre machine de développement.

### 📋 Prérequis

Assurez-vous d'avoir installé [Node.js](https://nodejs.org) (version 18+ recommandée) ainsi que votre gestionnaire de paquets `npm`.

### ⚡ Étapes d'installation

1. **Cloner le projet**

   ```bash
   git clone https://github.com
   cd amescao
   ```

2. **Installer les dépendances**

   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```

Le site est maintenant accessible localement à l'adresse : [http://localhost:3000](http://localhost:3000).

## 🚀 Scripts disponibles

Dans le dossier du projet, vous pouvez exécuter les commandes suivantes :

| Commande        | Description                                                                    |
| :-------------- | :----------------------------------------------------------------------------- |
| `npm run dev`   | Lance le serveur de développement local avec rechargement à chaud (Turbopack). |
| `npm run build` | Compile l'application pour la production (indispensable pour tester la PWA).   |
| `npm run start` | Démarre l'application compilée en mode production.                             |
| `npm run lint`  | Analyse le code pour trouver et corriger les erreurs de style ou de syntaxe.   |
